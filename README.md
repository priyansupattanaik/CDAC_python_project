# Online Quiz System

**Live Demo:** [https://priyansupattanaik.github.io/CDAC_python_project/](https://priyansupattanaik.github.io/CDAC_python_project/)

Python quiz app with performance analytics. Register, take quizzes, view charts. Admin can manage questions and view student stats.

**Admin login:** `admin` / `admin`

## Run Locally (Python)

```bash
pip install -r requirements.txt
python flask_app.py
```

Open http://localhost:5000

CLI version: `python main.py`

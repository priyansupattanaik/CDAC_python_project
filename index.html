CDAC Python Project

Online Quiz System

Performance analytics, student dashboards, admin controls — built with Python, Flask, and CSV storage.

View on GitHub Setup Guide

Features

👤

Student Portal

Register, login, take quizzes, and track your scores with charts and statistics.

📊

Performance Analytics

Line charts for score trends, pie charts for accuracy, and detailed attempt history.

🛡️

Admin Dashboard

Global leaderboard, per-student metrics, and question bank management.

📁

CSV Storage

No database required. Users, results, and questions stored in simple CSV files.

💻

Dual Interface

Run as a terminal CLI app or a full Flask web application.

✅

Input Validation

Username, password, and answer validation with clear error messages.

Setup & Run

1. Clone & install

git clone https://github.com/priyansupattanaik/CDAC_python_project.git
cd CDAC_python_project
python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux / Mac
source .venv/bin/activate

pip install -r requirements.txt

2. Run web app

python flask_app.py

Open http://localhost:5000 in your browser.

3. Run CLI app

python main.py
Admin login: username admin / password admin
Students must register a new account before taking quizzes.

How It Works

1. Register

Student creates an account. Data saved to users.csv.

2. Take Quiz

Questions loaded from questions.csv. Answers checked on submit.

3. Save Results

Score written to results.csv for analytics.

4. View Analytics

Charts generated with Matplotlib and shown on the dashboard.
